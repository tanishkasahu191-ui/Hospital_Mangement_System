**Hospital Management System**
----------------------------
A console-based hospital record management application using Python and SQLite.

---
**Overview of the Project**

- The Hospital Management System is a **Python-based application** designed to simplify and automate basic hospital operations.
- It allows the user to manage patient records,doctor details,and appointments using a simple **command interface**.
- The project uses **SQLite** as the backend database to ensure data is stored safely and permanently.

---

## ●  **Features**

- Patient Management

---->Add new patient

---->View existing patient records

---->Update patient details

---->Delete a patient


- Doctor Management

---->Add doctor

---->View doctors

---->Delete doctor


- Appointment Management

---->Creating a new appointment

---->View all appointments

---->Database Handling

----->Automatic table creation

---->Centralized database connection

---->Uses lightweight SQLite backend



----------------------------------------------------
## ● **Technologies / Tools Used**
- Programming Language: Python 3.14.0
  
- Database	SQLite (sqlite3 module)
  
- Code editor: Visual studio  Code
---------------------------------------

**Steps to Install & Run the Project**
- Create following files in the code editor
  
1.main.py

2.patients.py

3.doctors.py

4.appointments.py

5.database.py

(hospital.db will be created automatically once the program runs)

- Ensure Python is Installed

- Check Python version by running this command on terminal:
(python --version)

- Run the Project,navigate to the project folder:(HOSPITAL MANAGEMENT SYSTEM)

- Run the main script by running in vs code's terminal:
(python main.py)

- Once run, the menu appears:

1. Manage Patients
2. Manage Doctors
3. Manage Appointments
4. Exit


---------

## ● **Instructions for Testing**

You can test the system by performing the following:

1.**Test Patient Module**

- Add 2–3 patients

- View the list

- Verify they appear correctly

- Delete one and check again


2. **Test Doctor Module**

- Add doctor details

- Display doctor list

- Remove a doctor


3. **Test Appointment Module**

- Use an existing patient ID + doctor ID

- Schedule an appointment

- View appointment table
